Portfolio

Surf Betty Website

This was the first landing page that I built as a final project at the end of the SheCodes Basics workshop. Previous to this, I had no background or experience with coding. I applied my newfound knowledge of the foundations of HTML, CSS, and Javascript. As an avid surfer, I had a lot of fun working on this project that showcases something that I'm passionate about- creating a positive and vibrant female surfing community in Aotearoa (New Zealand).

Vanilla JS Weather App

This was the first interactive application that I built as a final project for the SheCodes Plus workshop. I built a responsive app that shows the weather forecast in any city in the world. This course taught advanced front-end technologies, advanced JavaScript techniques, how to integrate a live API, and tools such as Git and GitHub to facilitate a streamlined development workflow.

React JS Weather app

This was the final project in the SheCodes React workshop. I learned the fundamentals of React framework, which taught me how to build interactive user interfaces and web applications quickly and efficiently with significantly less code than with vanilla JavaScript. I was able to convert my earlier vanilla JS weather app to a React weather app.
